## 2. 스크립트 작업 전 구현 방법 고민
1. 플랜카드(요소) 클릭시, 클릭한 플랜카드를 찾아서 selected 클래스를 붙인다.
2. 그외 다른 플랜카드(plan-card를 가진 article태그)에 selected 클래스가 있다면, selected 클래스를 제거해준다.
3. selection-info 클래스를 가진 요소(div태그)를 찾아서 show 클래스를 붙여준다.
4. 내가 클릭한 요소(플랜카드)를 체크하여, 그 플랜카드의 이름자리(data-id="plan-price"값을 가진 태그)에 텍스트를 넣어준다.
