## 변수 · 상수 작성 과제 - 콤마(,) 사용으로 인한 문제

변수, 상수 작성 과제 중 5번, 7번 문제
변수를 작성시에 값 할당 부분에 콤마를 포함하여 값을 할당 하였다.


그랬더니 값이 정상 출력 되지 않았다.


자바스크립트에서 (,)는 숫자 구분자가 아니라 콤마 연산자가 되는것으로 알게되었다.
콤마를 사용하면 15,800점 앞의 15는 버리고 마지막값 800이 되어! 값은 800이 떨어질것이다.


그럼 여기서 궁금한거 하나 더!
그럼 콘솔창에 찍힌 값(ex: 15800) 이니! 컴퓨터에서 소수점을 파악하는가..?!


알아보니~ 소수점·콤마 자동 처리 안 하는걸로 파악!
콤마 있게 사용하려면 가공이 필요하다는 것을 알게 되었다.


## 변수 · 상수 작성 과제 - 변수 키워드 const 사용시 대문자, 소문자 사용방법(상수)

그리고 상수 선언할때 다시 한번 리마인드 하게 된 것!


상수라고 판단될때는 변수 키워드 const를 쓰는것에만 초첨을 맞췄는데..
변수명을 쓸때 대명사, 소문자 구분도 신경 써야 한다는걸 다시 리마인드하며 신경써서 작성 하였다.


대문자 const : 절대 안 바뀌는 값 + 설정값 느낌
소문자 const : 코드 흐름 안에서 한 번 정해지고 안 바뀌는 값
