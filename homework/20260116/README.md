# 과제 수행 과정 기록

## 1. 과제 초기 세팅
과제 작업을 하기위해 초기 세팅 하였다.  


초기세팅은 야무쌤이 수업시간에 제공해주신  
폴더구조와 파일들을 가져와서 사용하였다..

야무쌤 리소스중에서  
불필요한 자료들은 제거 또는 주석 처리를 하였다.


추후 작업을 진행하면서~  
필요한 리소스들은 추가하고, 불필요한 리소스들은 제거할 예정이다.


야무쌤 찬스를 사용하여 ㅎㅎ  
비교적 수월하게! 초기세팅을 마무리하였다.
